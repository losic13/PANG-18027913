# 미션 1: 난이도 및 규칙

[게임 룰 상세](./game_rule.md)에 정의된 PLAYER/BUBBLE/WIRE/BLOCK 공통 규칙을 그대로 따르되, 미션 1은 핵심 루프 검증을 위한 최소 난이도로 구성한다.

## 난이도 구성

- **Bubble 개수**: 시작 시 큰 Bubble 1개만 등장한다(추가 생성 없음).
- **분할 단계**: 큰 Bubble → 중간 Bubble 2개 → (중간 Bubble 각각) 작은 Bubble 2개까지만 분할되며, 작은 Bubble은 Wire에 맞으면 바로 소멸한다.
- **BLOCK**: 사용하지 않는다. 좌우 벽과 바닥으로만 구성된 단일 평면 스테이지.
- **파워업**: 제공하지 않는다.
- **WIRE**: 기본 1발만 지원하며 연사/강화 없음.

## 제한 조건

- **제한 시간**: 99초.
- **PLAYER 이동 속도**: 좌우 이동 속도는 기본값 1단계로 고정(가감속 없음).
- **PLAYER 목숨**: [게임 룰 상세](./game_rule.md)의 목숨 규칙을 따른다(Bubble에 닿을 때마다 하나씩 감소, 모두 소진 시 실패).
- **FPS**: 30 고정.

## 승패 조건 (요약)

| 조건 | 결과 |
|---|---|
| 화면 내 모든 Bubble(분할된 것 포함)을 제거 | 성공 |
| PLAYER가 보유한 목숨을 모두 소진 | 실패 |
| 99초 초과 | 실패 |

## 미션 1의 목적

파워업, 다중 스테이지, BLOCK, 여러 Bubble 동시 등장 등 복잡한 요소 없이 "조작 → Wire 발사 → 충돌 → 분할 → 승패 판정"의 핵심 루프가 정상 동작하는지 검증하는 것이 미션 1의 목표다. 이후 미션에서 Bubble 개수 증가, BLOCK을 활용한 스테이지 확장, 파워업 등이 추가될 수 있다.
