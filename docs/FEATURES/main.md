# 메인 화면 구성

## 목적

게임 진입 시 사용자가 가장 먼저 마주하는 화면으로, 플레이 시작 전 최소한의 안내와 진입 동선을 제공한다.

## 화면 구성 요소

- **타이틀**: 게임 이름(PANG)을 화면 상단 또는 중앙에 표시
- **시작(Start) 버튼**: 클릭 또는 스페이스 키 입력 시 게임 화면(미션 1)으로 전환
- **조작 안내**: 좌우 이동 키, 발사 키(스페이스) 등 기본 조작법을 간단히 안내
- **버전/미션 표시**: 현재 진행 가능한 미션(예: 미션 1)을 표기

## 동작 흐름

1. 앱 최초 진입 시 메인 화면이 렌더링된다.
2. 사용자가 시작 버튼을 누르거나 지정된 키를 입력하면 게임 화면으로 전환된다.
3. 게임 종료(성공/실패) 후 메인 화면 또는 결과 화면으로 돌아올 수 있는 진입점을 제공한다(상세 흐름은 추후 확정).

## 범위 밖

- 로그인/회원가입, 랭킹, 설정(사운드/난이도) 화면은 미션 1 단계에서 다루지 않는다.
