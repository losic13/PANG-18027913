# 게임 룰 상세

PANG 게임 전반에 적용되는 공통 규칙을 정의한다. 미션별 세부 난이도/수치는 [mission1.md](./mission1.md) 등 각 미션 문서에서 별도로 다룬다.

## PLAYER

- 키보드 좌우 방향키로만 이동하며, 점프 동작은 존재하지 않는다.
- 스페이스 키를 누르면 현재 위치에서 수직 위 방향으로만 Wire를 발사한다.
- Bubble에 닿으면 목숨을 하나 잃는다. 보유한 목숨을 모두 잃으면 게임이 실패로 종료된다.
- 화면 내 모든 Bubble을 소멸시키면 게임이 승리로 종료된다.

## BUBBLE

- 좌우 방향으로 공처럼 튀기며 이동한다.
- 중력의 영향을 받아 낙하하며, 바닥이나 Block에 튕길 때마다 직전과 같은 높이까지 다시 올라간다(에너지 손실 없이 일정한 높이를 유지).
- 좌우 벽이나 Block에 부딪히면 진행 방향이 반전된다.
- 화면 밖으로 나갈 수 없으며, 화면 경계에서 항상 반사된다.
- Wire에 맞으면 한 단계 작은 크기의 Bubble 2개로 분할된다.
- 가장 작은 크기의 Bubble은 Wire에 맞으면 분할되지 않고 소멸한다.
- 분할 직후 두 개의 자식 Bubble은 서로 반대 방향으로 튕겨 나가 시작한다.

## WIRE

- Space 키 입력 시 PLAYER의 현재 위치에서 수직 위 방향으로 발사된다.
- Bubble에 닿으면 해당 Bubble을 분할(또는 소멸)시키고 Wire는 사라진다.
- Block은 뚫지 못하며, Block에 닿으면 Bubble에 영향을 주지 않고 그 자리에서 사라진다.
- 화면 상단, Block, Bubble 중 무엇에든 먼저 닿을 때까지 직선으로 이동한다.

## BLOCK

- 부서지지 않는 고정 지형이다.
- Bubble은 Block 위로 튕겨 오를 수 있다(Block이 발판 역할을 한다).
- Wire는 Block을 통과하지 못하고 그 자리에서 소멸한다.

## 충돌 판정 요약

| 충돌 대상 | 결과 |
|---|---|
| Wire ↔ Bubble | Bubble 분할(또는 소멸), Wire 소멸 |
| Wire ↔ Block | Wire 소멸(Bubble에는 영향 없음) |
| PLAYER ↔ Bubble | PLAYER 목숨 1개 감소 |
| Bubble ↔ Block/벽/바닥 | Bubble 진행 방향 반전(튕김), 이전과 같은 높이로 재상승 |
| Bubble ↔ Bubble | 별도 처리 없음(서로 통과) |

## 승패 판정

- **성공**: 화면 내 모든 Bubble이 소멸하면 게임을 승리로 종료한다.
- **실패**: PLAYER가 보유한 목숨을 모두 소진하거나, 제한 시간을 초과하면 게임을 패배로 종료한다.

## 게임 루프 / 타이밍

- 게임 진행은 고정 FPS(초당 프레임)를 기준으로 갱신한다.
- 오브젝트별 이동/낙하 속도는 FPS를 바꾸지 않고 각 오브젝트의 speed 값 조정으로 처리한다.
- 제한 시간은 카운트다운 형태로 화면에 표시한다.
